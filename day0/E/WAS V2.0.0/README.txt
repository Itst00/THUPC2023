WAS V2.0.0 和 WAS for multiplayer V1.0.0 保留了几乎全部题面上的机制和数值。WAS V2.0.0 是一个从头开始的游戏。WAS for multiplayer V1.0.0 在创建角色时会提供满级角色和 40000 初始金钱，但只提供双人对战模式。

由于 Windows11 的终端设置可能与之前版本不一样，所以可能需要手动将终端大小设置为 38 行 120 列进行游戏。

WAS V2.0.0 使用 data.dll 作为玩家数据文件，level.dll 作为关卡配置文件。虽然不建议修改但也不反对修改。

WAS for multiplayer V1.0.0 使用 data1.dll 作为数据文件。编好队伍后，可将后手玩家的数据文件改名为 data2.dll 即可使用对战模式。该数据和 WAS V2.0.0 数据格式不相同，数据不互通。

数据文件不存在时会自动创建。所有的 .dll 文件都不是动态链接库。这是特性。

程序经过 WinXP→Win7→Win10→Win11 的更迭，所以可能会有少量没排除的显示问题。

如果想提出一些平衡性问题也欢迎找 E_Space 反馈。
